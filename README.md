### 제안배경
- 현재 우리나라의 장애인복지는 장애인 당사자만을 복지의 수요자로 설정하고 정책을 만들어왔다. 장애부모를 둔 비장애아동에게 학습권을 보장해주는 복지가 필요
- 2017년 장애인 실태조사』에 따르면, 장애부모는 자신의 장애로 인해 자녀 양육에 어려움이 있다고 응답
